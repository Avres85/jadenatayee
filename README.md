# Architecture Flipbook

Architecture flipbook built with Next.js, mounted directly to a static PDF file.

## Quick start

1. Install dependencies:
```bash
npm install
```

2. Run web app:
```bash
npm run dev
```

## Routes

- Viewer: `http://localhost:3000/viewer`

## PDF source

- The viewer serves `public/portfolio-compressed.pdf`.
- Replace that file to update the displayed portfolio.

## Useful scripts

- `npm run dev`
- `npm run build`
- `npm run lint`
- `npm run typecheck`
